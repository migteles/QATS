# QATS - Quality Assurance Tool Support for exploratory tests

Simple automation program made in Python with the purpose of assisting QAs during the process of exploratory tests.

This automation includes:
• EventLogger to collect both mouse movements and clicks alongside keyboard commands made during the test;
• Implemented Screen Recorder for proof of evidence made during testing;
• Basic GUI with timer.

This project was made as a Undergraduate Thesis for Computer Engineering course on FAMETRO University.
